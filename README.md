# 2025-2_Heros_Table
2025-2학기 알고리즘과게임컨텐츠 기말 프로젝트: 영웅의 식탁(게임)
- 제작: 놀고싶조 (9분반)

## 이 게임에 대해
세상은 균형으로 유지됩니다. 영웅은 그 균형을 지키는 인물이지요.

그렇지만 영웅에게도 균형이 필요한 것은 마찬가지입니다.

몬스터와의 싸움에서 잠시 벗어나 식탁에서 숨을 돌려봅시다.

- **게임 목표**: 탄수화물, 단백질, 지방의 영양 균형을 고려하여 최대한 오래 생존해봅시다.

## 조작 방법
- **좌로 이동:** `A`
- **우로 이동:** `D`
- **점프:** `스페이스 바`

## 게임 방법
### 1. 영양소 게이지
- 플레이어는 탄수화물, 단백질, 지방 3가지 영양소를 관리해야 합니다.
- 각 영양소는 0~200의 범위를 가지며, 100에서 시작합니다.
- 매 초마다 모든 영양소는 1씩 줄어듭니다.

### 2. 게임 오버
- 어느 하나라도 영양소 수치가 0이 되거나 200이 되면 즉시 게임이 종료됩니다.

### 3. 영양소 수치와 움직임
- 플레이어의 영양소 수치에 따라 캐릭터의 움직임이 변화합니다.
- **부족 (< 50):** 50 미만인 영양소 개수만큼 이동 속도와 점프 높이가 낮아집니다.
- **과다 (> 150):** 150 초과인 영양소 개수만큼 이동 속도와 점프 높이가 높아집니다.
